  <ul>

    <li><strong>Styling:</strong> Tailwind CSS with Daisy UI</li>

  <ul>
    <li>Practice building with Next.js</li>
    <li>Apply styling using Tailwind CSS with Daisy UI components</li>
    <li>Fetch data from JSONPlaceholder API</li>
  </ul>
